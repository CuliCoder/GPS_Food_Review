Flow chính của app

  • Auth: truy cập trang login → nhập user/pass → kiểm tra → trả token.
  • Guest: chọn ngôn ngữ → map → nearby venues → venue detail.
  • Vendor: login → payment VNPay → tạo POI → chờ admin duyệt.
  • Audio: map click → nearby venues → audio stream → auto play.
